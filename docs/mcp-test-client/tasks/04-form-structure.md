# Задача 04: Чтение формы и URI элементов

Статус: `todo`

## Цель
Добавить чтение формы по `form://{name}` и зафиксировать правила URI для адресации control внутри формы.

## Область
Входит:
- Ресурс-шаблон `form://{name}`.
- Снимок формы в JSON-контракте.
- Правила формирования `control://{form}/{name}`.
- Ошибки разрешения URI формы и control.

Не входит:
- Отдельное чтение control как ресурса.
- Взаимодействие с form-элементами.
- Нотификации об изменениях UI.

## Входные данные и допущения
- Поддержка resource template в MCP-компоненте считается доступной к моменту реализации.
- `form://{name}` строится из `ИмяФормы` с percent-encoding.
- `control://{form}/{name}` строится из `ИмяФормы` и `Имя` элемента.
- Если `Имя` элемента пустое, URI control не формируется.

## Подзадачи
1. Описать `form://{name}` в ресурсном контракте.
2. Реализовать разрешение формы по `ИмяФормы`.
3. Сформировать JSON-снимок формы с полями `type`, `uri`, `title`, `name`, `children`.
4. Зафиксировать правила генерации `control://{form}/{name}` для именованных элементов формы.
5. Обновить документацию так, чтобы `ui://active/form` и `ui://forms/{formId}` больше не использовались.

## Формат ответа
- `contents` — массив из одного элемента.
- `text` — JSON‑снимок формы.

## Обработка ошибок
- `UI_NOT_FOUND` при отсутствии формы по `form://{name}`.
- `UI_AMBIGUOUS_OBJECT` при нескольких формах с одним именем.
- `-32602` при некорректном URI формы.
- `-32603` при ошибке чтения формы.

## Проверки
- `form://{name}` возвращает снимок формы.
- Снимок формы содержит `uri`, `title`, `name`, `children`.
- Для именованных элементов формы формируются `control://{form}/{name}`.
- `ui://` URI форм больше не документируются и не публикуются.

## Критерии готовности
- Ресурс `form://{name}` возвращает форму по новому контракту.
- URI control сформированы по единым правилам.
- Ошибки разрешения формы и control соответствуют контракту.

## Риски и неизвестные
- Одно и то же `ИмяФормы` может встретиться у нескольких открытых форм.
- Не все элементы формы гарантированно имеют заполненное `Имя`.
