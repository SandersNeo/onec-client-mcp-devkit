## Tools

### `test_client_start`
Запускает процесс тест‑клиента 1С и подключается к нему.

Параметры:
- `command_line` — строка запуска 1С (обязательный).
- `working_dir` — рабочий каталог (опционально).
- `startup_delay_sec` — пауза перед подключением (опционально).
- `host` — хост для подключения (опционально).
- `port` — `TPort` для подключения (опционально, по умолчанию `1538`).
- `client_id` — идентификатор клиента (опционально).

Результат:
- `started` — булево.
- `connected` — булево.
- `details` — строка или структура.

### `test_client_controls`
Возвращает дерево элементов управления по текущему контексту.

Параметры:
- `scope` — `application|window|form`, по умолчанию `window`.
- `form_title` — заголовок формы (опционально).
- `form_name` — имя формы (опционально).
- `depth` — глубина дерева (по умолчанию 2).
- `timeout_sec` — таймаут поиска (опционально).

Результат:
- `root` — корневой элемент.
- `tree` — массив элементов.

### `test_client_click`
Имитирует нажатие или активизацию элемента.

Параметры:
- `scope` — контекст поиска.
- `type` — тип элемента (опционально).
- `title` — заголовок элемента (опционально).
- `name` — имя элемента (опционально).
- `form_title` — заголовок формы (опционально).
- `form_name` — имя формы (опционально).
- `timeout_sec` — таймаут поиска (опционально).

Результат:
- `clicked` — булево.
- `element` — описание найденного элемента.

### `test_client_open_form`
Открывает форму через командный интерфейс.

Параметры:
- `command` — текст команды (обязательный).
- `form_title` — ожидаемый заголовок формы (опционально).
- `timeout_sec` — таймаут ожидания (опционально).

Результат:
- `opened` — булево.
- `form` — описание формы.

### `test_client_input`
Вводит текст в поле через `ВвестиТекст(<Текст>)`.

Параметры:
- `text` — текст для ввода (обязательный).
- `name` — имя поля (опционально).
- `title` — заголовок поля (опционально).
- `form_title` — заголовок формы (опционально).
- `form_name` — имя формы (опционально).
- `timeout_sec` — таймаут поиска (опционально).

Результат:
- `input` — булево.
- `field` — описание поля.

## Resources
- `ui://active/window` — активное окно.
- `ui://active/form` — активная форма.
- `ui://forms/{formId}` — конкретная форма.
- `ui://elements/{elementId}` — элемент управления.

## Уведомления
- `notifications/ui/changed` — изменения UI.
- `notifications/tools/listChanged`.
- `notifications/resources/listChanged`.
- `notifications/prompts/listChanged`.
