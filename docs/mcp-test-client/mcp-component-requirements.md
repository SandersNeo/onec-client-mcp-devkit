# Требования к MCP компоненте (AddIn.WebTransport.mcp)

## Базовые требования
- Поддержка JSON‑RPC сообщений MCP без `id` (уведомления).
- Возможность отправки уведомлений от сервера к клиенту.
- Стабильное соединение с подтверждением отправки.

## Требования для уведомлений
- API для отправки `notifications/*`.
- Возможность пакетной отправки уведомлений (debounce).
- Ограничение частоты отправки.

## Требования к маршрутизации
- Сохранение контекста сессий MCP.
- Разделение каналов для запросов и уведомлений.

## Диагностика
- Детальные логи отправки уведомлений.
- Метрики задержек и ошибок доставки.

## Заглушки
- Названия методов компоненты для отправки уведомлений.
- Возможность подписки клиента на типы уведомлений.
