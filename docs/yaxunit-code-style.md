# YAxUnit Code Style (наблюдения)

Документ фиксирует фактический стиль BSL по исходникам
Это не официальный стандарт, а срез повторяющихся паттернов.

## Структура файла
- Декларации переменных чаще всего вынесены в область `#Область ОписаниеПеременных`.
- Экспортируемые API обычно в `#Область ПрограммныйИнтерфейс` или `#Область СлужебныйПрограммныйИнтерфейс`.
- Внутренние процедуры и функции находятся в `#Область СлужебныеПроцедурыИФункции`.
- Обработчики событий выделяются в `#Область Обработчики...`.
- Каждая область закрывается `#КонецОбласти`.

## Атрибуты и препроцессор
- Контекстные атрибуты `&НаКлиенте`, `&НаСервере` размещаются над объявлением.
- Компиляционные ветки используют `#Если`, `#ИначеЕсли`, `#Иначе`, `#КонецЕсли`.
- Локальные подавления проверок встречаются рядом с конструкторами и возвратами: `//@skip-check <check-id>`.

## Именование
- Модули и объекты имеют префикс `Мсп_`, PascalCase, чаще кириллица.
- Процедуры и функции названы глаголами, PascalCase.
- Переменные и параметры — PascalCase, осмысленные имена.
- Булевы признаки читаются как утверждения: `ЕстьФильтр...`, `Включено`, `ЭтоФайловаяБаза`.

## Форматирование
- Табовая индентация, единый отступ на блок.
- Каждая инструкция заканчивается `;`.
- Пустые строки разделяют смысловые блоки.
- Длинные логические выражения разбиваются на строки, `И` и `ИЛИ` стоят в начале строки.
- Цепочки вызовов форматируются переносом строк с ведущей точкой.
- Конструкторы структур пишутся в одну строку, ключи задаются строкой с запятыми: `Новый Структура("Ключ1, Ключ2", ...)`.

## Доккомментарии
- Однострочное описание, затем пустая строка `//`.
- Секции `Параметры:` и `Возвращаемое значение:` (для функций).
- Формат параметров: `Имя - Тип - описание`.
- Альтернативные типы перечисляются отдельными строками.
- Для ограничений и примеров используются `Ограничения:` и `Пример:`.
- Ссылки на связанные методы/типы в кратком описании оформляются через `см.`.

### Шаблон
```bsl
// Краткое описание.
//
// Параметры:
//  ИмяПараметра - Тип - Описание
//
// Возвращаемое значение:
//  Тип - Описание
```
